Estructura de datos (paralelo C2)

****SISTEMA DE GESTION ACADEMICA*****
1. Agregar un nuevo alumno
Registra un nuevo alumno en el sistema.
Parámetros:
  - `ID`: Un número entero único.
  - `Nombre`: Texto con el nombre del alumno.
  - `Apellido`: Texto con el apellido.
  - `Carrera`: Nombre de la carrera.
  - `Fecha de Ingreso`: Formato "dd/mm/yyyy".

2. Buscar alumno por su ID
Muestra la información de un alumno usando su ID.
Parámetro:
- `ID`: Número entero correspondiente al ID del alumno.

3. Buscar alumno por nombre
Lista todos los alumnos con el nombre especificado.
Parámetro:
- `Nombre`: Texto con el nombre a buscar.

4. Borrar un alumno
Elimina un alumno y sus inscripciones del sistema.
Parámetro:
- `ID`: Número entero del ID del alumno.

5. Crear un curso nuevo
Registra un nuevo curso en el sistema.
Parámetros:
- `ID`: Número entero único.
- `Nombre`: Texto con el nombre del curso.
- `Capacidad máxima`: Número entero.
- `Carrera`: Nombre de la carrera.
- `Profesor`: Texto con el nombre del profesor.

6. Buscar curso por ID
Muestra la información de un curso usando su ID.
Parámetro:
- `ID`: Número entero del ID del curso.

7. Buscar curso por nombre
Lista todos los cursos con el nombre especificado.
Parámetro:
- `Nombre`: Texto con el nombre a buscar.

8. Eliminar un curso
Elimina un curso y sus inscripciones asociadas.
Parámetro:
- `ID`: Número entero del ID del curso.
